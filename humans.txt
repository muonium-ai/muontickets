Install MuonTickets as a git submodule under `tickets/mt/muontickets`.

Quick setup commands (run from your project root):
git submodule add https://github.com/muonium-ai/muontickets.git tickets/mt/muontickets
git submodule update --init --recursive --depth 1
uv sync
uv run python3 tickets/mt/muontickets/muontickets/mt.py init

See full onboarding + ticket-template workflow in README.md.
